## Stack

- Express (web framework)
- MongoDB (NoSQL db)
- Joi (validation)

## Requirements

- NodeJS (tested on v10)
- Yarn
- MongoDB

## Usage

Edit `.env.sample` and rename to `.env`

Development

- Run `yarn watch` or `yarn debug`

For testing, run `yarn test` or `yarn test:coverage`

Production

  1. Run `yarn build`
  2. Run `yarn start`
